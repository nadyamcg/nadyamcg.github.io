hello, world.

the blog is static: a simple posts.json file lists posts, and each post is a plain html file.

how to add a post:

  1. copy this file as hello-world-old-web.html and edit.
  2. add a new entry to posts.json with title, slug, date, and optional summary.
  3. commit and push. the blog index updates automatically.
